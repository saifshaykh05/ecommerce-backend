 E-Commerce Backend

This is a backend project built using Spring Boot for an E-commerce system.  
It provides REST APIs for users, products, and orders.

---

 Features

- User registration and login
- JWT authentication
- Role-based access (Admin & User)
- Product management
- Order management

---

 Technologies

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL

---

 API Examples

- POST /auth/register  
- POST /auth/login  
- GET /products  
- POST /products  
- GET /orders  

---

 Purpose

This project shows backend development skills using Spring Boot and real-world concepts.
